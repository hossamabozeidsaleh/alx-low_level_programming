# alx-low_level_programming


C is an imperative procedural language, supporting structured programming, lexical variable scope and recursion, with a static type system. It was designed to be
compiled to provide low-level access to memory and language constructs that map efficiently to machine instructions, all with minimal runtime support.



it contains : 4 directory 

 `0x00-hello_world`

 `0x01-variables_if_else_while`

 `0x02-functions_nested_loops`
 
 `0x03-debugging`
 
 `0x04-more_functions_nested_loops`

 `0x05-pointers_arrays_strings`
